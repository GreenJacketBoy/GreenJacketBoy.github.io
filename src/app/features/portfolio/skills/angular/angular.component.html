Angular, what I never knew I wanted

Angular est un framework de front end (c'est-à-dire d'interface utilisateur), facilitant principalement la création d'applications web dynamiques. Il offre des outils et une structure de développement constante, ce qui en fait un bon choix pour beaucoup d'entreprises.

Une application web réalisée avec Angular tourne entièrement sur la machine cliente, le serveur ne s'occupe que d'envoyer les fichiers HTML, CSS, Scripts, et autres médias demandés par le client.

Cela permet de créer une application web relativement complexe, tout en minimisant la charge que doit supporter notre serveur. Par exemple, le site web sur lequel vous êtes actuellement a été réalisé avec ce Framework, et il est hébergé sur GitHub Pages, qui ne supporte pas de script côté serveur.

Je l'ai découvert et principalement utilisé durant mon stage de deuxième année, mais aussi, comme dit précédemment, pour mon site personnel.
