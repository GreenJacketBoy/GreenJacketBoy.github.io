La Shei-quoi ?

Ce projet a pour but principal de me faire développer une application avec ReactJS. Son but secondaire est de recréer la Tablette Sheikah du jeu Zelda : BOTW, ainsi que certaines de ses fonctionnalités, comme :

  • Une carte interactive dans le même style graphique, avec la possibilité d'ajouter différents points d'intérêt (voire capture d'écran en jeu).

  • Un gestionnaire de fichiers (uniquement des images dans le jeu), avec la possibilité d'y associer une description de ce qu'elle représente.

Cette application est donc réalisée avec React JS, auquel je n'avais jamais touché avant, et la librairie MapLibre GL que j'ai utilisée durant mon stage de 2025, et qui gère l'affichage de la carte et ses interactions. J'aurais techniquement pu utiliser Angular, mais j'avais envie d'apprendre React afin de voir s'il y avait une réelle raison à sa popularité.

Le projet est actuellement en développement, donc cette page est sujette à de futures modifications. Cependant, le projet est déjà très bien avancé avec :

  • une application ergonomique au possible (fidèle au jeu)
  • une application stylisée (carte, style et décoration, tout fait à la main sans IA)
  • l'ajout, la modification et la suppression de markers
  • des markers de couleur unique (un par couleur maximum)
  • des markers à symboles illimités
