La Shei-quoi ?

Ce projet a pour but principal de me faire développer une application avec ReactJS. Son but secondaire est de recréer la Tablette Sheikah du jeu Zelda : BOTW, ainsi que certaines de ses fonctionnalités, comme :

  • Une carte interactive dans le même style graphique, avec la possibilité d'ajouter différents points d'intérêt (voire capture d'écran en jeu).

  • Un gestionnaire de fichiers (uniquement des images dans le jeu), avec la possibilité d'y associer une description de ce qu'elle représente.

Cette application est donc réalisée avec le framework React et la librairie MapLibre GL, que j'ai utilisée durant mon stage de 2025, et qui gère l'affichage de la carte et ses interactions. Le projet est actuellement en développement, donc cette page est sujette à de futures modifications.

Pour plus d'informations : https://github.com/GreenJacketBoy/Sheikah-Slate-Web
