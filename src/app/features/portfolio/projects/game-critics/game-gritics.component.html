Projet Game-Critics

Dans les grandes lignes, les consignes pour ce projet étaient les suivantes :

Réaliser une application web permettant aux utilisateurs d'émettre des critiques sur une instance d'un thème que nous aurions choisis. Dans notre cas, nous avons choisis de réaliser une application de critique de jeux vidéos.

L'important détail était les contraintes sur les méthodes et technologies que nous devions utiliser, à savoir :

  • Le BackEnd devait consister d'une Base de Donnée SQL, avec laquelle nous pouvions intéragir au travers d'une API un utilisant le Framework PHP Symfony
  • Le FrontEnd devait être réalisé à l'aide du Framework VueJS, et se devait de présenter au mieux les possibilitées offertes par la réactivité caractéristique de ce framework

Ayant déjà eu de l'expérience sur d'autres framework réactifs tels que React et Angular, j'ai principalement travaillé sur le FrontEnd du projet, même si j'ai contribué à un nombre non-négligeable de résolution de bugs et d'incohérence minimes dans l'API.

(P.S. Le projet étant toujourrs en cours au moment de la rédaction, le site web affiché sur les captures d'écran n'est pas exactement esthéthiquement plaisant, mais il est proche de 100 % fonctionnel)
